xm-demos-rap

Minimal Node.js Project

Welcome to xm-demos-rap!

This is a simple Node.js environment ready for forking and custom development.

Getting Started

  • Modify the index.js file to build your application
  • Add dependencies to package.json as needed
  • Fork this project for your own custom development

Project Structure

  • index.js - Main entry point
  • public/ - Static files directory
  • public/base.css - Base styles
  • public/js/ - JavaScript files
  • public/img/ - Image assets
